# Economic-Data-Analysis- POWER BI DASHBOARD

An interactive Power BI dashboard built to analyze and compare economic indicators across countries using the Cost of Living Index 2022 dataset.

## What the Dashboard Covers
-KPI Cards — Avg Cost of Living, Avg Groceries, Avg Rent, Avg Restaurant Price, Avg Purchasing Power

-Ribbon Chart — Cost of living trends across countries

-Column & Bar Charts — Country-wise comparison of cost of living and economic indicators

-Clustered Column Charts — Side-by-side breakdown of multiple economic metrics by country

-Slicer — Dynamic country-level filtering for custom analysis

## Tools Used

-Microsoft Power BI

-Data Cleaning & Transformation (Power Query)

## Key Learnings

-Understood how cost of living varies across countries and what drives purchasing power

-Practiced building KPI-driven dashboards for economic data interpretation

-Developed skills in data visualization and storytelling using real-world datasets
